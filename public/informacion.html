Información del Sistema

Este sistema de información permite monitorear y evaluar diferentes parámetros a través del tiempo. Los datos mostrados incluyen mediciones de temperatura, humedad, presión y el factor FPF.

El sistema utiliza un código de colores para indicar el estado de las mediciones:

  • Verde: Estado normal
  • Amarillo: Alerta preventiva
  • Naranja: Alerta moderada
  • Rojo: Alerta crítica

Los datos son actualizados regularmente y pueden ser visualizados tanto en formato tabular como en gráficos para un mejor análisis de tendencias.
